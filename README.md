평가 항목	상세 선정 근거
① 데이터 처리 역량(업무 부합도)	
• 본 과업은 "사용자가 데이터를 직접 제공 → 모델이 가공"하는 방식임.
• 제미나이 3.1 Pro는 3사 모델 중 유일하게 갭 계산(2.0억→2.7억) 및 상승률(12%) 검산을 완벽하게 수행하여, 본 과업의 핵심인 '데이터 가공 역량'에서 가장 뛰어남.
② 지시 이행 및 마찰 최소화	
• 타 모델(클로드 Opus 4.8)은 임의로 제목을 변경하고 페르소나를 수용하지 않아 매번 결과물을 재편집해야 하는 마찰 비용이 발생함.
• 반면, 제미나이는 요청한 비즈니스 보고서 형식을 그대로 반환하여 반복 업무 효율을 극대화함.
③ 환각(Hallucination) 방어력	
• 데이터가 부족한 부분에 대해 "정확한 실거래가 확인 필요"를 자발적으로 명시함.
• 타 모델(GPT-5.5)처럼 데이터에 없는 지역 특성을 임의로 추정하여 서술하지 않아 합격선 이상의 방어력을 보임.
